HUES Goes

HUES Goes is a way for HUES members to get together, plan events, and create community. We post events that we find that we will be attending, as well as events that we create that we would love others to come to. These could be social events, panel discussions, lectures, or “field trips” to interesting places. The purpose of HUES Goes is to give people a group to come to these events with, as we all know that it can be difficult to get a group together. So check the HUES Goes Facebook page and look for the latest event info!
